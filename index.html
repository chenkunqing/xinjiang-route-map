🗓️ 9天行程总览
南疆帕米尔 + 北疆伊犁环线

南疆第一段：帕米尔高原（6/2-6/4）

喀什古城 → G314高原线 → 白沙湖 → 喀拉库勒湖 → 塔什库尔干 → 盘龙古道 → 班迪尔蓝湖 → 喀什 ✈️ 乌鲁木齐

北疆第二段：伊犁环线（6/5-6/9）

乌鲁木齐 → 赛里木湖 → 果子沟 → 霍城薰衣草 → 霍尔果斯 → 特克斯 → 喀拉峻 → 唐布拉 → 独库公路 → 乌鲁木齐

⛽ 三个加油盲区（务必加满！）
  • G314高原段（喀什→塔县）：中间无加油站
  • 唐布拉段（新源→乔尔玛）：100km无加油站
  • 独库北段（乔尔玛→独山子）：150km无加油站

🫀 高原反应自查 ▼
常见症状
  • 头痛、胸闷、气短
  • 失眠、食欲减退、疲倦
  • 嘴唇/指甲发紫
应对措施
  • 吸氧（氧气瓶/车载制氧机）
  • 喝葡萄糖水/红景天口服液
  • 动作放慢，避免剧烈运动
  • 保证睡眠，不要熬夜
  • 必要时及时就医
💡 提醒塔县海拔3100m，喀拉库勒湖3600m，盤龙古道4200m。到达当天不要洗澡！
🚗 自驾必知规则 ▼
  1. 加油站规则：驾驶员出示身份证+驾驶证，所有乘客必须下车站到安全区，后备箱打开检查。每次多花3-5分钟。
  2. 区间测速：高速和国道到处都是区间测速！严格按限速牌开。
  3. 安检站：减速停车出示身份证，有时检查后备箱。外地车可能被多问两句，正常配合。
  4. 时差：新疆用北京时间，但实际晚约2小时。9点天亮，22点天黑。饭店14:00才开门。
  5. 车辆：宋Pro插混，市区用电，高速/山路/独库全程靠油。独库和唐布拉无充电桩。
  6. 手机信号：城市/高速有信号。唐布拉、独库部分路段可能断网。离线地图必备！
